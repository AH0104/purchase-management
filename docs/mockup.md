## 仕入支払管理システム モックアップ概要

### 1. レイアウト共通コンセプト

- **サイドナビ固定**: 左側に 80px 幅のサイドバー、freee やスマレジ風のアイコンナビ。ホバーまたはクリックで 240px まで展開し、ラベルを表示。
- **主要メニュー**: ホーム（ダッシュボード）、納品データ、アップロード、仕入先、分析（将来）、設定。下部にヘルプとログアウト。
- **ヘッダー**: サイドバーの右側上部に検索バー、通知ベル、ユーザードロップダウンを配置。
- **メインコンテンツ**: 余白 24px のカードベース。背景は淡いグレー、カードは白。
- **カラーパレット**: サイドバーにネイビー、アクセントにティール、成功はグリーン、警告はアンバーを想定。

```
┌────┬──────────────────────────────┐
│ 🏠   │ ヘッダー: [検索バー            ]  🔔  ユーザー名 ▼ │
│ 📄   ├──────────────────────────────┤
│ ⬆️   │                                      │
│ 👥   │        メインコンテンツ              │
│ 📊   │                                      │
│ ⚙️   │                                      │
└────┴──────────────────────────────┘
```

### 2. ダッシュボード画面

- **サマリカード**: 今月仕入総額、未処理件数、今月支払予定、未消込件数を 2×2 グリッドで配置。アイコングラフィック入り。
- **仕入先別集計カード**: 左側に円グラフ、右側にランキングリスト。上部に「分析ページへ」リンク。
- **月次トレンド**: 折れ線グラフをカードで表示。将来的な分析拡張を見越し、期間フィルタ付き。
- **直近アップロード**: テーブル（アップロード日時、仕入先、件名、ステータス、担当者）。行クリックで詳細をサイドパネル表示。
- **FAB（Floating Action Button）**: 画面右下にアップロードアイコン付きボタン。

```
┌───────┬───────┐
│サマリ1  │サマリ2  │
├───────┼───────┤
│サマリ3  │サマリ4  │
└───────┴───────┘

┌──────────────┬──────────────┐
│ 仕入先別集計（円） │ 月次トレンド（折れ線） │
└──────────────┴──────────────┘

┌────────────────────────────┐
│ 直近アップロード一覧（テーブル）            │
└────────────────────────────┘

        ●（FAB: アップロード）
```

### 3. ファイルアップロードフロー

- **ステップ1: ファイル選択**
  - 仕入先選択ドロップダウン
  - ファイルドロップゾーン（PDF, Excel, CSV サポート）
  - 次へボタン
- **ステップ2: 解析結果プレビュー**
  - 抽出ヘッダー情報（納品日、伝票番号、合計等）
  - 明細テーブル（インライン編集可、行追加/削除）
  - エラー/警告メッセージ領域
- **ステップ3: 保存確認**
  - 登録内容サマリ
  - 保存ボタン、戻るボタン

```
ステップバー: [1 ファイル選択] → [2 データ確認] → [3 保存]

[仕入先 ▼]   [ファイルをドラッグ＆ドロップ]

解析結果
┌────────────┬────────────┐
│ 納品日:2025/10/15 │ 伝票番号:TEST-001 │ ...
├────┬────────┬────┬────┬────┐
│行番 │商品コード   │商品名 │数量 │単価 │金額 │
├────┼────────┼────┼────┼────┼────┤
│  1  │A001        │トマト │10   │500  │5,000│
└────┴────────┴────┴────┴────┴────┘
```

### 4. 仕入先管理画面

- **一覧ビュー**
  - 上部に検索バーとステータスフィルタ（アクティブ／非アクティブ）。
  - テーブル列: 仕入先名、コード、支払条件、担当者、連絡先（メール）、ステータス。
  - 行末に編集／削除アクション。
  - 右上に「新規登録」ボタン。
- **ドロワーフォーム**
  - 右側からスライド表示するドロワー。銀行口座情報は除外。
  - フィールド: 仕入先名、コード、支払条件（日数）、担当者、メール、電話番号、住所、メモ、ステータス切り替え。
  - 保存／キャンセルボタンを下部に固定。

### 5. 納品データ一覧・検索

- **フィルターバー**: 期間、仕入先、ステータス、フリーワード。
- **テーブル**: 納品日、伝票番号、仕入先、合計、ステータス、最終更新。
- **詳細サイドパネル**: 行クリックで右側からスライド表示、ヘッダー情報＋明細タブ。
- **エクスポートアクション**: テーブル上部に CSV/Excel 出力ボタン。

```
フィルター: [期間 ▼] [仕入先 ▼] [ステータス ▼] [キーワード]

┌────┬────┬────┬────┬────┬────┐
│日付 │伝票番号│仕入先│合計 │状態 │最終更新│
└────┴────┴────┴────┴────┴────┘

詳細パネル
┌────────────┐
│ 納品書詳細                │
├────────────┤
│ヘッダ情報 …             │
│明細タブ …               │
└────────────┘
```

### 6. メッセージとローディング

- ローディング: スピナー＋「解析中…」など日本語メッセージ。
- エラーダイアログ: 赤系トーストで詳細説明と再試行ボタン。
- 成功通知: 緑系トースト「登録が完了しました」。
